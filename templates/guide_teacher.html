1. Créer un sujet d'examen

1

Accéder à la gestion des sujets

Dans le menu principal, cliquez sur Sujets d'Examens. Vous verrez la liste de vos sujets existants et le bouton + Nouveau Sujet.

2

Rédiger ou téléverser le sujet

Deux méthodes disponibles :

  • Téléversement : importez un fichier PDF ou Word — l'IA analyse automatiquement le contenu et génère un barème de notation correspondant.
  • Génération IA : choisissez un EC (Élément Constitutif) et cliquez sur "Générer des Suggestions d'Examen avec IA". Sélectionnez une suggestion pour générer un examen complet avec questions numérotées et points.
Après création, un aperçu du sujet ET du barème généré s'affiche pour validation avant utilisation.
3

Vérifier la séparation questions / barème

Le barème de notation (qui contient les réponses) est stocké séparément. Les étudiants ne voient jamais le barème pendant l'examen.

2. Créer et configurer un examen en ligne

1

Ouvrir la création d'examen

Dans Examens en Ligne, cliquez sur + Créer un Examen. Remplissez le formulaire :

  • Titre de l'examen, sujet associé
  • Date/heure de début et de fin
  • Durée (en minutes)
  • Instructions optionnelles
2

Configurer la sécurité anti-triche

Ajustez les paramètres de surveillance selon vos besoins :

Changements de fenêtre

Nombre max avant bannissement automatique

Visage absent

Seuil de détections "aucun visage" avant alerte

Copier/Coller

Activer/désactiver le presse-papier

Clic droit

Bloquer le menu contextuel

Outils Dev

Bannir si outils développeur détectés

3. Activer l'examen

1

Passer de "Planifié" à "En cours"

Sur la carte de l'examen, cliquez sur le bouton Activer. Le statut passe à En cours. Les étudiants peuvent maintenant composer pendant la plage horaire définie.

Si un étudiant clique sur "Composer" avant l'heure, il voit une alerte avec la date et l'heure exactes de début.
2

Ouvrir le Dashboard de Surveillance

Cliquez sur Surveiller sur la carte de l'examen. Le dashboard s'ouvre dans un nouvel onglet avec les flux vidéo en direct de tous les étudiants.

4. Dashboard de surveillance en temps réel

Le dashboard affiche une grille de toutes les tentatives actives avec vidéo en direct de chaque étudiant.

Flux caméra HD

Vidéo en direct de chaque étudiant (caméra + détection faciale)

Partage d'écran

Bouton Voir écran quand l'étudiant partage son écran — vue plein format

Score de risque

0–100% calculé en temps réel selon les incidents détectés

Envoyer un avertissement

Message affiché immédiatement sur l'écran de l'étudiant avec popup orange

Envoyer un message

Communication directe avec l'étudiant (popup bleue sur son écran)

Messages des étudiants

Les étudiants peuvent vous écrire depuis leur panneau "Contacter l'enseignant"

Exclure un étudiant

Bannissement immédiat avec saisie de la raison — irréversible

Logs d'activité

Historique complet des événements pour chaque étudiant

Les étudiants à risque élevé (score ≥ 70%) apparaissent avec un encadré rouge dans la grille pour les repérer rapidement.

5. Corriger les copies avec l'IA

1

Clôturer l'examen

Cliquez sur Clôturer sur la carte de l'examen. Les étudiants encore en session sont soumis automatiquement.

2

Accéder aux copies

Rendez-vous dans Copies d'Étudiants. Sélectionnez les copies soumises et lancez la correction IA. L'IA utilise le barème du sujet pour noter chaque réponse avec un feedback détaillé.

3

Réviser et valider

Vérifiez les notes proposées par l'IA, ajustez si nécessaire, puis publiez les résultats. Une notification email est envoyée aux étudiants.

6. Accéder aux enregistrements

1

Snapshots caméra automatiques

Depuis le dashboard, cliquez sur le bouton Enregistrements en haut à droite. L'onglet "Snapshots caméra" affiche toutes les captures d'écran horodatées prises automatiquement pendant l'examen.

2

Vidéos enregistrées (REC)

Si vous avez activé le bouton REC sur une carte étudiant pendant l'examen, la vidéo est sauvegardée sur le serveur. L'onglet "Vidéos enregistrées" génère des liens de téléchargement valables 2 heures.

Les liens d'accès expirent après 2 heures. Cliquez sur "Actualiser" pour en obtenir de nouveaux.

7. Publier les notes et relevés

1

Saisie / import des notes

Allez dans Notes. Les notes des copies corrigées sont automatiquement associées. Vous pouvez aussi importer en masse via CSV ou saisir manuellement.

2

Générer les relevés

Depuis Relevés de Notes, générez les relevés semestriels officiels (PDF) avec calcul automatique du GPA et crédit ECTS.
